How I Built My Portfolio Site

Published: May 1, 2025

Building my portfolio website was an exciting journey where I combined aesthetics with simplicity. I started by drafting the layout using HTML5 and styled it using CSS3 with responsiveness in mind. A major focus was the clean dark/light mode toggle using CSS variables and JavaScript.

I chose to host the project on GitHub Pages for its ease and speed. Additionally, I used FontAwesome for icons and ensured the entire site is accessible and mobile-friendly. One of the best parts was structuring the blog to scale in the future, which you're reading now.

I plan to continue enhancing the site with JavaScript interactivity, animations, and maybe even some React components as I grow.
